# azure-apim-graphql

This repository contains Azure API Management GraphQL infrastructure and samples.

## Contents

### GitHub GraphQL Sample Application

A Python command-line application demonstrating how to use GitHub's GraphQL API with personal access token authentication.

**Location:** `github-graphql-sample/`

**Features:**
- View authenticated user information
- List user repositories
- Get detailed repository information
- Direct GraphQL API interaction with GitHub

For setup and usage instructions, see [github-graphql-sample/README.md](github-graphql-sample/README.md)
